# Business Calculator

A sleek, local-first web application for calculating Cost of Goods Sold (COGS) and managing Bill of Materials (BOM) recipes, featuring an Apple-inspired Liquid Glass UI.

## Features
- **Local Storage**: All data is securely stored on your local browser.
- **Master Database**: Manage ingredients, volumes, and base cost.
- **Multisheet Excel Export/Import**: Full system serialization out and in!
- **Dynamic Calculation**: Instantly updates Operations, Tax, and Profit margins.
